# Andres — Portfolio Personal

Sitio web personal donde comparto mi camino hacia Data Analyst, mis proyectos de análisis de datos y reflexiones sobre inversión.

## Stack

- HTML5 semántico
- CSS3 + Bootstrap 5
- JavaScript vanilla

## Estructura
mi-portfolio/

├── index.html        # Home

├── about.html        # Sobre mí

├── projects.html     # Proyectos

├── blog.html         # Blog

├── contact.html      # Contacto

├── css/              # Estilos

├── js/               # Scripts

└── assets/img/       # Imágenes

## Desarrollo local

Cloná el repo y abrí `index.html` en tu navegador. No requiere build ni servidor.
